# Skills

A collection of CLI skills for automation and productivity.

## Available Skills

| Skill | Description |
|-------|-------------|
| [video-clipper](./video-clipper/) | Download and transcribe videos from TikTok, YouTube, or Instagram Reels to Obsidian notes |

## Installation

Each skill has its own directory with setup instructions. Generally:

```bash
cd <skill-name>
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
